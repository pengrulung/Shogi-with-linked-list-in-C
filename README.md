# Getting Started!

## Compile & Run & Play

```sh
# Compile
cd 109503510_assignment_2/
gcc -o main src/main.c inc/move.h inc/init.h inc/print.h inc/rule.h 
# Run
./main -n -s [filename.txt]
# Play(數字1~9)
動哪個(段) (筋):
去哪裡(段) (筋):
輸入f繼續玩 or 輸入b反悔 or 輸入q離開:
# FOR EXAMPLE
動哪個(段) (筋):3 1
去哪裡(段) (筋):4 1
輸入f繼續玩 or 輸入b反悔 or 輸入q離開:f

# 重播棋譜
./main -l [filename.txt]
```
